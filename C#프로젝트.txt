PMS 프로젝트

요구사항 분석

1. 프로젝트 진행상황 진행률(%) 표시 
- 해당 프로젝트의 각 화면과 화면에 해당하는 기능들에 대한 목록들을 수치화 하여 진행률을 표시
- 진행중인 프로젝트, 완료된 프로젝트 분리 가능
- 진행중인 프로젝트 강제완료 기능
- 프로젝트들의 바로 가기 연결 기능


2. 진행중인 프로젝트 상세보기 화면
- 프로젝트의 내의 각 항목들을 상세히 볼 수 있다.
	- 레벨로 표시
		-> 최상위레벨 : 해당 프로젝트 -> 다음레벨 : 화면 항목들 
		-> 다음레벨 : 화면에 해당하는 각 기능들 -> 다음레벨 : 각 기능들에 대한 세부 항목들
	- 화면항목에서 아이콘 클릭시 화면의 스토리보드 확인가능
- 프로젝트의 상세 내용을 확인할 수 있다.
	- 프로젝트의 기간, 비용, 인원, 목적, 기술스택, 소요시간, 시작날짜, 팀 프로젝트인지 개인 프로젝트인지, 
	  팀 프로젝트일경우 포지션별로 맡은 항목 나열
	- 각 기능별로 현재 이슈상황 확인 가능
	- 각 기능별 우선순위 확인 가능


3. 프로젝트 첫 등록 기능
- 개인 프로젝트일 경우 | 팀 프로젝트일 경우
	- 요구사항을 단순화 | 담당자 별로 등록 한다.
- 화면항목을 등록하는 기능
- 각 화명항목 우선순위 설정가능


4. 프로젝트 일지 등록 기능
- 개인 프로젝트일 경우 | 팀 프로젝트일 경우
	- 개인일 경우 단순화
	- 팀 프로젝트일 경우 담당자 별로 배분 받은 항목에 진척도 등록
	- 각 기능별 우선순위 설정 가능
